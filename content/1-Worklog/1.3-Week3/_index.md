### Week 3 Objectives:

* Build a strong understanding of AWS compute services, with a focus on Amazon EC2 and Auto Scaling.
* Explore AWS storage solutions and learn how backup and disaster recovery are implemented in the cloud.
* Complete hands-on labs using AWS Backup and AWS Storage Gateway.
* Deploy a static website with Amazon S3 and improve content delivery using Amazon CloudFront.
* Learn advanced Amazon S3 features for data protection and replication.

### Week 3 Achievements:

* Acquired practical knowledge of Amazon EC2 by learning how instances, AMIs, EBS volumes, User Data, and Metadata work together within AWS.
* Developed a clear understanding of Auto Scaling and its role in improving application availability and resource utilization.
* Successfully created backup plans, executed restore operations, and understood the importance of automated data protection with AWS Backup.
* Learned how AWS Storage Gateway connects on-premises environments with AWS storage services through a hybrid cloud architecture.
* Published a static website on Amazon S3 and enhanced its accessibility and performance using Amazon CloudFront.
* Explored advanced Amazon S3 capabilities, including Versioning and Cross-Region Replication, to improve data durability and availability.
* Followed AWS cost management best practices by deleting all lab resources after completing each exercise.

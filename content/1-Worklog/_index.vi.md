**Trong trang này**, em tóm tắt nhật ký công việc hằng tuần trong quá trình tham gia chương trình AWS First Cloud Journey. Worklog ghi lại các nội dung đã học, các bài lab đã hoàn thành và cách kiến thức được áp dụng trong suốt quá trình thực tập.


Chương trình được thực hiện trong 12 tuần, bao gồm các nội dung về AWS fundamentals, security, storage, database, analytics, developer tools và dự án cuối kỳ AWS Serverless Event Portal. Nội dung từng tuần được tóm tắt như sau:

**Tuần 1:** [Làm quen với AWS và các dịch vụ cơ bản trong AWS](1.1-week1/)

**Tuần 2:** [Thực hành IAM, MFA, AWS Budgets và các kiến thức cơ bản về AWS Support](1.2-week2/)

**Tuần 3:** [Học compute và storage với EC2, Backup, Storage Gateway, S3 và CloudFront](1.3-week3/)

**Tuần 4:** [Tìm hiểu AWS storage services, AWS Backup, S3 storage classes và lifecycle management](1.4-week4/)

**Tuần 5:** [Thực hành Amazon S3 nâng cao, VM Import/Export và AWS Storage Gateway](1.5-week5/)

**Tuần 6:** [Học AWS security fundamentals, IAM Roles, Security Hub, Lambda automation và resource management](1.6-week6/)

**Tuần 7:** [Thực hành IAM quotas, AWS KMS encryption, CloudTrail logging và Athena auditing](1.7-week7/)

**Tuần 8:** [Học IAM policies nâng cao, AWS Secrets Manager và nền tảng AWS database services](1.8-week8/)

**Tuần 9:** [Triển khai Amazon RDS và thực hành database migration với AWS DMS và Amazon Aurora](1.9-week9/)

**Tuần 10:** [Tìm hiểu AWS analytics services, Amazon DynamoDB, CloudShell và Cloud9](1.10-week10/)

**Tuần 11:** [Phát triển backend cho AWS Serverless Event Portal](1.11-week11/)

**Tuần 12:** [Cải thiện tích hợp Event Portal và kiến trúc AWS](1.12-week12/)
